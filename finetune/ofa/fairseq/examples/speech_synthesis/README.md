Speech Synthesis (S^2)
===

Speech synthesis with fairseq.

- Autoregressive and non-autoregressive models
- Multi-speaker synthesis
- Audio preprocessing
- Automatic metrics
- Similar data configuration as [S2T](../speech_to_text/README.md)
